# AI Learn · AI 学习力教练

> 面向青少年学习力教练的 AI 赋能培训体系 —— 用 ima 知识库 + Copilot 构建「学习档案 + 智能分析」双引擎工作流。

## 项目定位

为学习力教练提供一套可复用的 AI 工具培训方案，覆盖学情诊断、动力激发、方法指导、习惯养成、家长沟通、考试冲刺六大场景。

## 技术栈

- **PPT 引擎**：基于 [Guizang PPT Skill](https://github.com/op7418/guizang-ppt-skill) 的瑞士国际主义风格
- **交付格式**：单文件 HTML，浏览器直接打开，支持键盘翻页 / 触屏滑动
- **设计语言**：Swiss Style · IKB 克莱因蓝 · Inter + Noto Sans SC

## 使用方式

```bash
# 克隆仓库
git clone https://github.com/YOUR_USERNAME/ailearn.git

# 直接用浏览器打开任意培训 PPT
open training/ai-coach-ima/index.html
```
